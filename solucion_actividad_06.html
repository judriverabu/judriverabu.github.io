Representacin de funciones univariadas

Grfica de las Funciones:

  • \(f(x) = x^{3} + x^{2},x\hspace{1mm}\epsilon\hspace{1mm}\mathbb{R}\)
  • \(f(x) = e ^{(-2x)},x\hspace{1mm}\epsilon\hspace{1mm}\mathbb{R}\)
  • \(f(x) = x \cos(\alpha x),-10 \leq x \leq 10\)
  • \(f(x) = |x| , -1 \leq x \leq 1\)
  • \(f(x) = 1/|x| , -1 \leq x \leq 1\)
  • \(f(x) = x^{2} , -1 \leq x \leq 1\)
  • \(f(x) = \frac{sin(x)}{x}, -1 \leq x \leq 1\)

Se procede a realizar las graficas de las funciones propuestas, para esto, se tiene en cuenta el rango de \(x\):

curve(x^3+x^2 ,from = -10, to = 10, las = 1, 
      xlab = expression(x), 
      ylab = "",
      lwd= 2,ylim=c(-1000,1000))
curve(exp(-2*x),add = TRUE, col=2, lwd = 2)
grid()
legend("bottomright",
       legend=c(expression(f(x)==x^3+x^2),
                expression(f(x)==exp(-2*x))),
       col= c(1,2), lwd = 2, bty = "n")

Las funciones anteriores:

\(f(x) = x^{3} + x^{2},x\hspace{1mm}\epsilon\hspace{1mm}\mathbb{R}\) ,
\(f(x) = e ^{(-2x)},x\hspace{1mm}\epsilon\hspace{1mm}\mathbb{R}\)

Son continuas, no presentan saltos ni interrupciones en su linea resultante, adicional, presentan una curva suave.



curve(abs(x) ,from = -1, to = 1, las = 1, 
      xlab = expression(x), col=7,
      ylab = "",
      lwd= 2,ylim=c(-2,2))
curve(x^2,add = TRUE, col=3, lwd = 2)
curve(sin(x)/x,add = TRUE, col=4, lwd = 2)
grid()
legend("bottomright",
       legend=c(expression(f(x)== abs(x)),
                expression(f(x)== x^2),
                expression(f(x)== frac(sin(x),x))),
       col= c(7,3,4), lwd = 2, bty = "n")

En el intervalo \(-1 \leq x \leq 1\) :
La funcin \(f(x) = |x|\), es continua, no presenta saltos ni interrupciones de linea
La funcin \(f(x) = \frac{sin(x)}{x}\) es discontinua, presenta salto, no est definida en el punto 0.

La funcin \(f(x) = x^2\),es convexa.


curve(1/abs(x) ,from = -1, to = 1, las = 1, 
      xlab = expression(x), col= 6,
      ylab = "",
      lwd= 2,ylim=c(-6,30))
legend("bottomright",
       legend=c(expression(f(x)==1/abs(x))),
       col= 6, lwd = 2, bty = "n")

La funcin \(f(x) = 1/|x|\) en \(-1 \leq x \leq 1\) es discontinua (compuesta por dos asinttas).

Representacin de funciones de dos variables

Para la representacin en dos variables se tomar la funcin de Rosenbrock: \(f(x_1,x_2) = 100(x_2 - x_1 ^2) + (1-x_1)^2 , x_i \hspace{1mm}\epsilon\hspace{1mm} [-2.048,2.048] , i=1,2\), el cuel alcanza su valor mnimo en \(x_1 = 1\) y \(x_2 = 1\)


x1 <- seq(-2.048,2.048, length.out = 50)
x2 <- seq(-2.048,2.048, length.out = 50)
X <- expand.grid(x1,x2)
f1 <- function(x1,x2){
  return(100*(x2-x1^2)+((1-x1)^2))
}
z <- f1(X[,1],X[,2])
Z <- matrix(z, ncol = 50, nrow = 50)
contour(x1, x2, Z)

persp(x1, x2, Z,d = 2, theta = 15)
